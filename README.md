# Админ-панель - Агентство новостроек

React админ-панель для управления проектами новостроек.

## Технологии

- React 18
- Vite
- Tailwind CSS
- React Router

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Админ-панель будет доступна по адресу http://localhost:5174

## Сборка

```bash
npm run build
```

## Функциональность

- Список всех проектов
- Создание нового проекта
- Редактирование существующего проекта
- Удаление проекта
- Управление всеми данными проекта (цены, характеристики, изображения, инфраструктура)
